## install instructions

1. clone
2. run `npm install`;
3. run `npm start`;

## explanation

very simple 5e point buy calculator using standard rules as written.

Default Point Buy Rules
*    Available Points - 27
*    Maximum Attribute Before Racial Bonus - 15
*    Minimum Attribute Before Racial Bonus - 8
